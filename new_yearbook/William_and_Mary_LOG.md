## 四、VPN 下载方案

当 VPN 可用时，下载命令：

```bash
# 单册下载示例
wget "https://archive.org/download/colonialecho190103coll/colonialecho190103coll.pdf" \
  -O "William and Mary 1901.pdf"

# 全量下载脚本（循环 1901-1930，跳过 1904）
for year in $(seq 1901 1930); do
  [ $year -eq 1904 ] && continue
  vol=$((year - 1898))
  id="colonialecho${year}$(printf '%02d' $vol)coll"
  echo "Downloading $year (Vol $vol, $id)..."
  wget -q --show-progress "https://archive.org/download/${id}/${id}.pdf" \
    -O "William and Mary ${year}.pdf"
done
```

或使用项目中 **已有的** `scripts/safe_download.py`（含 tqdm + PDF 完整性校验）。
